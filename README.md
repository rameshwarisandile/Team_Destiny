## Project Overview

- **Project Name:** SafeVoice – *Anonymous Reporting & Support Platform*

### Overview
**SafeVoice** is a **secure**, **AI-powered**, anonymous reporting platform designed to help victims of **harassment** report incidents safely and access real-time **emotional**, **legal**, and **community support**.

It leverages **AI detection**, strong **encryption**, and verified partnerships with **NGOs**, **counselors**, and **law enforcement** so each report gets the right help quickly while keeping the user’s **identity private**.

## Core Features
1. **Anonymous reporting** without login or personal information.
2. **AI-based urgency detection** and **case categorization**.
3. **Encrypted chat support** with verified counselors.
4. **Secure proof uploads** for video, image, and document files.
5. **Real-time support connection** with verified NGOs, police, and legal advisors.
6. **End-to-end encryption** for all data and communication.
